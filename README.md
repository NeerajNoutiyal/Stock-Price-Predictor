# 📈 Stock Price Predictor

A Machine Learning project that predicts stock prices using historical stock market data.

## Features

- Historical stock data
- Data visualization
- Machine Learning prediction
- Linear Regression model

## Technologies

- Python
- Pandas
- NumPy
- Scikit-Learn
- Matplotlib
- yfinance
